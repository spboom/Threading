Items Ineligible for Return

  • Opened software*
  • Electronic Software Downloads
  • Software Up-to-Date Program Products (software upgrades)
  • Apple Store Gift Cards
  • Apple Developer Connection Products (Membership and Technical Support)
  • Apple Print Products

* You can return software as long as it hasn't been installed on any computer. Software that contains a printed software license cannot be returned if the seal or sticker on the software media package is broken.

About Returns

  • We supply a free, prepaid shipping label to make it easy to ship your items to us.
  • You have 14 calendar days from the time your items are delivered to initiate a return.
  • You can return almost any product purchased on the Apple Online Store. Not sure about yours? See a list of ineligible items.
  • For a replacement of a damaged or defective product, please contact Apple Store Customer Service at 1-800-MY-APPLE.
